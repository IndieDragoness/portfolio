Over the course of six years I've developed various scripts, assets, and art for the Unity Game Engine. Some of these works I've sold on the Unity Asset Store. Others are longer term hobby projects. Machine Learning Agents in Unity Game Engine

As part of my coursework for my masters degree program I wrote a paper and developed a demonstration game that attempts to teach Reinforcement Learning Procedural Content Generation (RL-PCG). In the demonstration game, I utilized Unity's Machine Learning Agents framework to bring Tensorflow capabilities into the game engine.

Bolt Visual Scripting

While learning Unity, I created a series of tutorials for Hobby Game developers on the Bolt Visual Scripting language. In addition, I created and submitted a public C# module that let users bridge data between Bolt and Behavior Designer.

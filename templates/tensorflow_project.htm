As part of my work as a Senior Software Engineer, I developed a data flow framework for the company both for training and for production. The framework supported object detection in images, using a Convolutional Neural Network (CNN). In addition, I made changes to our Tensorflow codebase to support transfer learning, a method of building off of an existing neural network to support new features without restarting training completely from scratch.

TensorFlow is an end-to-end open-source platform for machine learning. It has a comprehensive, flexible ecosystem of tools, libraries and community resources that lets researchers push the state-of-the-art in ML and developers easily build and deploy ML powered applications.
